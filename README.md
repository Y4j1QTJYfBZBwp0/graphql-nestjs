# Microservices with Nestjs - Training.

## Description

Repository containing my code generated during my training and studies about GraphQL + NestJS.

## Tech Stack

* VSCode
* GitHub Copilot
* GitHub Copilot Labs
* Nestjs

## Installation

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
